11OVC Trigger — скрипт расшифровок

Интерактивный компонент в шапке. Никакой "настоящей" расшифровки нет — все варианты равнозначны. Это и есть пасхалка.

Демо — кликни на ovc.me
ovc.me: Оптимизация Векторных Цепочек : Машинная Экспертиза

Серьёзные расшифровки появляются в ~85% случаев. Пасхалки — в ~15%. Мета-пасхалка — на 10-м клике подряд.

Формула расшифровки
[СТРУКТУРА] [O][V][C] : [M][E]

Каждая буква домена = слово. Двоеточие делит на префикс (ovc) и суффикс (.me). Русский вариант: О-В-Ц : М-Э/Е.

[RAG] Open Vector Context : Model Environment

Фокус на векторах, контексте, валидации, машинном обучении.

[TOC] Optimized Value Chain : Management Execution

Фокус на цепочках ценности, узких местах, метриках, ограничениях.

[BIM] Operational Virtual Construction : Master Engineering

Фокус на виртуальном строительстве, объектах, визуализации, инжиниринге.

Полный массив — все расшифровки

<script>
const expansions = [ /* массив выше */ ];
let clickCount = 0;

function updateExpansion() {
  const el = document.getElementById('ovc-expansion');
  if (!el) return;
  clickCount++;
  if (clickCount === 10) {
    el.textContent = "Означает Всё Что Захочешь : Можешь Ещё Раз";
    clickCount = 0; return;
  }
  const showEaster = Math.random() < 0.15;
  const pool = showEaster ? expansions.slice(12, 19) : expansions.slice(0, 12);
  el.textContent = pool[Math.floor(Math.random() * pool.length)];
}

document.addEventListener('DOMContentLoaded', () => {
  updateExpansion();
  document.getElementById('ovc-trigger')
    .addEventListener('click', updateExpansion);
});
</script>

Работает без зависимостей. Подходит для Tilda, WordPress, Webflow — вставить как HTML-блок.
